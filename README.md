# My-Fifth-SSMS-Project


This is currently my largest SQL Server Management Studio (SSMS) project completed so far. It combines multiple SQL techniques including data exploration, data cleaning, transformations, joins, transactions, and schema modifications within a single workflow.
Throughout this project, I used several different SQL command types and functions, including:

Data Exploration
SELECT
SELECT *
Numeric Functions
ROUND()
CEILING()
Aggregation
COUNT()

NULL Handling
ISNULL()
COALESCE()
NULLIF()
String Functions
LOWER()
SUBSTRING()

Date & Time Functions
DATEDIFF()
GETDATE()
CONVERT()
Transactions
BEGIN TRANSACTION
COMMIT
ROLLBACK

Joins
INNER JOIN
LEFT JOIN
RIGHT JOIN
FULL JOIN
Type Conversion
TRY_CAST()

Database & Schema Modifications
CREATE DATABASE
ALTER TABLE
ALTER COLUMN
DROP CONSTRAINT
Metadata Queries
INFORMATION_SCHEMA.COLUMNS

These commands were used to practice real-world database tasks such as analyzing datasets, cleaning data, modifying schemas, and performing relational joins across tables.
